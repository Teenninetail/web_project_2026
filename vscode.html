VS code installment and guide

introduction

VS code is a text editor that make coding easier with the build in copilot and can be link to github for easy upload of your code.

Download guide

  1. First step: go to the official website Visual Studio Code.
  2. Second step: run the download installer and follow the instruction on the screen.
  3. third step: download git and run git config for your name and email.
